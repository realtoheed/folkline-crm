# Folkline CRM

A responsive CRM dashboard concept with a visual sales pipeline, designed for small teams to track deals from lead to closing.

## Features

- **Sales Pipeline** — Four deal stages: Qualified → Proposal → Negotiation → Closing
- **Searchable Deal Board** — Find deals instantly with stage-wise totals
- **One-Click Progression** — Move deals between stages with a single click
- **Add Deal Workflow** — Inline modal with status feedback
- **Responsive Sidebar** — Collapsible navigation with mobile support
- **Accessible** — Keyboard navigation + `prefers-reduced-motion` support

## Tech Stack

- **Framework:** Next.js 16 (React 19)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Font:** Geist (via next/font)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Status

Frontend concept — focused on UI/UX and responsive design. Designed as a foundation for backend integration (database, auth, real-time sync).
