# Folkline CRM

A responsive, open-source CRM dashboard concept built with Next.js, React, and TypeScript.

## Features

- Sales pipeline with four deal stages
- Searchable deal board and stage totals
- Add-deal workflow with inline feedback
- One-click deal progression
- Responsive sidebar and mobile navigation
- Accessible controls, focus states, and reduced-motion support

## Run locally

```bash
npm install
npm run dev
```
